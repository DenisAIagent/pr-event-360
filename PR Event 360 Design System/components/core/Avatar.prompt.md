**Avatar** — circular identity for journalists, media contacts and team members. Shows an image when `src` is set, otherwise up-to-2-letter initials on a soft tint.

```jsx
<Avatar name="Camille Roy" size={40} tone="blue" />
<Avatar name="Le Monde" src="/logo.png" size={48} />
```

Tones: `navy` (default), `blue`, `slate`. Keep ≥32px for legibility.
