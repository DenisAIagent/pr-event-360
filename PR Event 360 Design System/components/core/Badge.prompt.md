**Badge** — compact status pill. Tones map to the charter: `success` = Accrédité, `warning` = À relancer, `danger` = Refusé / Annulé, `info` = VIP / Média clé, plus `neutral` and `navy`.

```jsx
<Badge tone="success" dot>Accrédité</Badge>
<Badge tone="warning">À relancer</Badge>
<Badge tone="danger">Annulé</Badge>
<Badge tone="info">VIP</Badge>
```

Use `dot` for live/status contexts. Keep labels to 1–2 words.
